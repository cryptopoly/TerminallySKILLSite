Everything you need in one workspace

Stop juggling terminals, scripts, and SSH sessions. TerminallySKILL brings it all together.

Visual Command Builder

Auto-discovers every CLI tool on your PATH and presents them as form-based builders with flag selection, live preview, and zero syntax memorization.

Workflow Scripts

Build multi-step automation with placeholder variables, approval checkpoints, retry logic, and per-step error handling. Share flows as .tvflow files.

Full Terminal Sessions

Powered by xterm.js with split panes, command queue, output snapshots with diff viewer, in-terminal regex search, and ghost suggestions.

SSH & Encrypted VNC

SSH key setup helper, interactive remote shells, and encrypted VNC remote desktop streaming through SSH tunnels — no firewall rules needed.

AI-Powered Tools

Command review, explain, natural language drafting, and output analysis. Supports OpenAI, Anthropic, Gemini, Groq, Mistral, Grok, DeepSeek, Ollama, LM Studio, and more.

Projects

Organize work with named projects, each with their own working directory, environment variables, favorite commands, terminal sessions, and color badges.

Runs & Logs

Unified time-ordered log browser with full-text search, structured workflow run history, run comparison, and AI-powered log review.

Snippets & Themes

Reusable command templates with parameterized variables. 8 built-in themes (4 dark, 4 light) plus command palette, auto-updates, and full customization.

File Browser

Built-in directory viewer with syntax-highlighted file preview, line numbers, and quick navigation — no need to leave the workspace.
